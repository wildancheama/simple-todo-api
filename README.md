# Simple Todo API

This project is a simple REST API built with Node.js and Express.
I created this project while learning backend development with Node.js and Express
and building my understanding of how REST APIs work.

## Tech Stack
- Node.js
- Express
- Nodemon
- dotenv

## Getting Started

### Setup & Run
```bash
npm install
npm run dev
```

---

## The server will start at:
http://localhost:3000

---

## Environment Variables
Create a `.env` file in the root directory:
```env
PORT=3000
```
---

## Endpoint
- `GET /health` → Check if the server is running correctly
- `GET /todos`  → Get all todo items
- `POST /todos` → Create a new todo item
